Introduction/Background

Predicting house prices is critical to the real estate industry, as it provides valuable insight for buyers, sellers, and even investors. Since we have access to housing data, we want to build a model that takes into account various home characteristics, such as square footage and home type, for accurate and reliable home price predictions.

Studies such as by Kok et al. (2017) utilized a linear regression model to predict home prices with features including house size, number of rooms, and location. The model was proven accurate based on the given training dataset, and the results were easy to interpret.

We will be using the Ames Housing Dataset, an alternative to the popular Boston Dataset found in James et al. (2018), but with more features and observations. This dataset can be found at the following link: Ames Housing Dataset

Problem Definition

Problem

The problem we are addressing is the challenge of accurately predicting house prices located in Ames, Iowa. Traditional methods often rely on simple models or manual predictions which don’t capture the more complex relationships between predictive variables. This leads buyers, sellers, and investors to make suboptimal decisions because of a lack of accurate information.

Motivation

By accurately automating house price prediction, it would help homebuyers, sellers, real estate agents, and investors make informed decisions.

Methods

Data preprocessing methods

Handling missing data

To ensure that we don’t have any missing values, we’ll use an imputation technique where we replace the missing value with the mean or median. We can implement this using sklearn.impute.SimpleImputer().

Feature Scaling

Certain features such as house size or lot area can vary a lot in scale, and this can affect models specifically for linear regression. Hence, we’ll apply standardization using StandardScaler() from scikit-learn.

Log-scaling Target Price

We understand that there will be a heteroskedasticity problem with the dataset. When true home prices increase, we expect model error to increase as well. By performing a log transform on our target home prices, we can consider error in a more relative sense, removing the problem of overweighting observations with large true prices and underweighting others (Manning 1998).

Results and Expectations

The quantitative metrics we hope to fulfill by the end of this project are:

  • Achieve a sklearn.metrics.r2_score of > 0.85
  • Reduce the sklearn.metrics.mean_squared_error to < $20,000
  • Achieve a sklearn.metrics.mean_absolute_percentage_error of < 10%

The goal of this project is to develop an accurate and reliable model that can forecast house prices in Ames, Iowa, using a variety of house characteristics as explanatory variables.

As priorly mentioned, accurate price prediction would contribute to helping both buyers and sellers make better decisions by ensuring that properties are priced fairly and competitively. A more efficient real estate market can contribute to more sustainable urban development, as money is allocated more efficiently and responsibly.

Based on the prior research outlined, based on similar models, our model is expected to meet or exceed the performance goals. By incorporating a diverse range of features, the linear regression model will be capable of capturing and analyzing key patterns in data.
